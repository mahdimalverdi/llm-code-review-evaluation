## Framework Overview

Introduce the framework layers:

```text
Input/context quality
→ Generated comment quality
→ Problematic comment type
→ Usefulness and actionability
→ Mitigation decision
→ Cost and workflow impact
→ Evaluator validity
```

## Layer 1: Input and Context Quality

Define relevance, completeness, specificity, consistency, freshness, reviewability, provenance, behavioral evidence, attention load, and cost.

## Layer 2: Generated Comment Quality

Define correctness, grounding, relevance, specificity, explanation quality, usefulness, and actionability.

## Layer 3: Problematic Comment Type

Connect taxonomy labels to evaluation decisions.

## Layer 4: Mitigation Decision

Discuss show, suppress, rewrite, and escalate.

## Layer 5: Trade-Off Analysis

Discuss error reduction, useful-feedback preservation, review coverage, human effort, computational cost, and workflow impact.

## Layer 6: Evaluator Validity

Discuss human annotation and LLM-as-a-Judge as measurement instruments.

## Optional Specialized Sublayers

- secure review;
- static-analysis-guided review;
- context consistency;
- non-functional quality.
